Конфиг информации о правилах

| Поле            | Тип      | Описание                                                                 |
| --------------- | -------- | ------------------------------------------------------------------------ |
| `categoryID`    | `string` | Уникальный идентификатор категории                 |
| `ruleTitle` | `string` | Ключ для локализации названия категории. Обычно выглядит как `#STR_...`. Либо прямой перевод |
| `ruleIcon`  | `string` | Путь к иконке. Поддерживается формат `.edds`.                  |
| `rulesList`  | `array string` | Массив строк с правилами.    |
